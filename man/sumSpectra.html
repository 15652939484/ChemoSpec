Function to summarize a Spectra object

Description

Provides a summary of a "Spectra" object, essentially a more spectroscopist-friendly version of str().

Usage

sumSpectra(spectra)

Arguments

spectra An object of S3 class "Spectra"

Details

Prior to summarizing, chkSpectra is run with confirm = FALSE. If there are problems, warnings are issued to the console and the summary is not done.

Value

None.

Warning

The computation of frequency resolution does not currently take into account any regions removed from the middle of the range and hence will be in error.

Calls

chkSpectra

Called by

none
