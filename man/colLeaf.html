colLeaf {unknown}R Documentation

Function to color the leaves of a dendrogram when directly or indirectly analyzing a Spectra object

Description

Utility function called by HCA and ScoresHCA via dendrapply. Not intended for end-users.

Usage

colLeaf(n, spectra)

Arguments

n The leaf number (an integer).
spectra An object of S3 class "Spectra".

Value

None. The leaf colors of the relevant dendrogram object are modified to correspond to those in spectra

Called by

HCA, ScoreHCA

Note

The basic idea was found in the help archives. I can't write this kind of stuff!
