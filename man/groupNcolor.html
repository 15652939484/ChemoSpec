groupNcolor {unknown}R Documentation

Function to assign group membership and corresponding colors in the construction of a Spectra object

Description

A utility function which looks for gr.crit in the file names of .csv files and assigns group membership. Also assigns a color to each group. Not intended for users.

Usage

groupNcolor(spectra, gr.crit = NULL, gr.cols = c("auto"))

Arguments

spectra An object of S3 class "Spectra". Until this function acts on "Spectra" it is not quite complete.
gr.crit As per getManyCsv
gr.cols As per getManyCsv

Value

A complete object of S3 class "Spectra". Until this function has done its job, an object of class "Spectra" will not pass checks (see chkSpectra).)

Called by

getManyCsv
